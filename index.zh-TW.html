關於我

我是一名全端開發者，擅長多種技術。

在前端方面，我會運用 React.js、TypeScript、Vite 和 Next.js 等現代工具。這個首頁就是使用 Vite 建置的。

在後端方面，我精通 Go、Rust、PHP 和 Node.js 等程式語言。

我在 Zeabur 等 PaaS 解決方案上有豐富經驗，能確保應用程式在這類平台上順暢運作。此外，我熟悉 AWS（EC2、SES）、GCP（GCE）、Cloudflare（R2）和 DigitalOcean（Droplet、Docker registry）等雲端服務供應商。我也擅長使用 Git，並熟悉業界標準的貢獻流程和提交規範，如 Conventional Commits，以及主要使用 GitHub Actions 的 CI/CD 流程。我也參與開發以 Docker 為基礎的建置方案 Zeabur Pack，並也對 Kubernetes（Zeabur 基礎設施的核心）的使用尚算熟悉。

我偏好雲端原生和可擴展的解決方案，也在多個專案中運用了微服務架構和技術，例如：

  • 高科大智商系物聯網專案（Go，2024/06）：使用 Go 編寫，運用了 RabbitMQ、GraphQL、gRPC、OpenTelemetry、mTLS、PostgreSQL 等技術。
  • 高科大智商系物聯網專案（Rust，2024/12）：使用 Rust 編寫，運用了 S3（R2）、GraphQL、NATS、PostgreSQL 等技術。完全透過 NATS 進行通訊。
  • 資料庫練功房：主要為單體架構，但有把部分服務拆分為 HTTP 服務。這套系統原先採用微服務架構，但後來改用 Symfony 以便快速開發和原型驗證。

我持續探索軟體工程的最佳實踐和可擴展解決方案，來維持程式碼的高品質。歡迎與我分享您的看法！

此外，我也以業餘身分使用 Swift 或 React Native 為 Apple 生態系統開發應用程式。我有開發 macOS 和 iOS 應用程式的經驗，例如 String Catalog 合併工具，並協助改善一些 Swift 專案的並行處理。

在大型語言模型（LLMs）領域，我熟悉包括 GPT、Claude、Gemini 和 Deepseek 等各種模型，了解它們的最佳使用場景和 API 整合。我參與了 資料庫練功房 這個研究專案，運用 GPT 協助學生除錯 SQL 查詢。我的開源 LLM 專案包括 AI 字幕校正、自動化軟體翻譯和 以 OpenAPI tooling 為基礎的 Docker Compose 到 Zeabur 範本轉換。

除了開發，由於我修過智慧商務系眾多和資料分析相關的課程，我能使用 Python 進行資料分析。我亦設計一些作品，可以在 我的 Figma 社群檔案 中查看，雖然我不怎麼更新我的作品集。
